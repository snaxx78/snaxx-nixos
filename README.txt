# Hyprland shortcuts (AZERTY friendly)

## Touche principale
SUPER = touche Windows

---

## Applications
SUPER + Enter → terminal  
SUPER + D → launcher  
SUPER + E → fichiers  
SUPER + B → navigateur  

---

## Fenêtres
SUPER + Q → fermer  
SUPER + F → fullscreen  
SUPER + Shift + F → vrai fullscreen  
SUPER + V → floating  
SUPER + P → pseudo  
SUPER + S → split  

---

## Navigation (flèches)
SUPER + ← → gauche  
SUPER + ↓ → bas  
SUPER + ↑ → haut  
SUPER + → → droite  

---

## Déplacement fenêtres
SUPER + Shift + flèches  

---

## Resize
SUPER + Ctrl + flèches  

---

## Workspaces

⚠️ Clavier AZERTY :
- touche & = workspace 1
- é = workspace 2
- " = workspace 3
- etc

SUPER + 1..0 → aller workspace  
SUPER + Shift + 1..0 → envoyer  

---

## Navigation workspace (laptop)
SUPER + TAB → suivant  
SUPER + Shift + TAB → précédent  

SUPER + ` → dernier workspace  

---

## Screenshots
Print → zone  
Shift + Print → écran  
Ctrl + Print → fenêtre  

---

## Souris
SUPER + clic gauche → déplacer  
SUPER + clic droit → resize  

---

## Système
SUPER + X → lock  
SUPER + Shift + R → reload  
SUPER + M → exit  

---

## Important

👉 Les workspaces se créent automatiquement  
👉 Tu n’as rien à config pour ça
